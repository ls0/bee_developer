Developers Tools: The Chrome SnappySnippet



How to Get the Most Out of SnappySnippet


SnappySnippet is a Chrome Extension that is added to the "Inspect element", which pops up by right clicking on a browser page. After clicking SnappySnippet you can inspect and manipulate HTML, CSS and JavaScript with CodePen, JS Bin or jsFiddle after clicking the “Create a snippet from inspect element” and “Send” buttons, respectively. Once the code has been loaded into the sandbox, you can copy code that is useful, or you can manipulate code to see how things work. The sandbox is also a great place to learn html, css and javascript, and he sandbox can be used for sharing code with others by creating code, saving code and then sharing the code's url. In the above example we are examining code from the Coursera course list. Programmers at stackoverflow.com seem to prefer to give js Fiddle url links to show code ideas and examples. It is a useful learning tool to be able to see the html, css, js and output all on the same page. Thus, jsFiddle saves to the cloud and the used later by the originator or shared with others.


Summary of Steps


  1. 1. Find a website that interests you.
  2. 2. Right click on "Inspect element."
  3. 3. Click on "Element" and find interesting elements.
  4. 4. Click on "SnappySnippet" (furthest right from "Element").
  5. 5. Click on "Create a snippet from inspected element."
  6. 6. Click one of the three "Send" buttons at the bottom of page.
  7. 7. Play with code.
  8. 8. Learn about different elements.
  9. 9. The sandboxes are a good place to develop your own code.
  10. 10. Send url of saved code to others.

Going Further with Chrome


Firefox has been my browser of choice since I migrated away from Internet Explorer many years ago. However, since taking the Web Application Architectures course I took a closer look at Chrome. In many ways it appears that the Chrome browser is geared more for the web developer than Firefox. There are numerous add-ons for developers in the Chrome Store. So far only the SnappySnippet has really caught my attention. This tool is from Konrad Dzwiel.
